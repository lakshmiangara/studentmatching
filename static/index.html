Student Matching System


Room 404: Lakshmi Angara, Zima Patel, Alexander Infante, Jared Peterson



The scope of the project is limited to grouping compatible students with similar or closely matching preferences. The project will account student technical skills, his/her project preference and time of availability along with synchronous and asynchronous communication. System will make sure no more than 3 teams can work on the same project.
